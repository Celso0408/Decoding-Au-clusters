# Reference Frameworks

This directory contains the upstream repositories used for verifying the DimRedPy framework's correctness and parameter alignment.

These repositories are NOT part of the DimRedPy runtime. DimRedPy uses pure-Python equivalents (openTSNE, umap-learn, pymbar) to avoid C++ compilation dependencies.
